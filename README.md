# Simple Social Network

A simple social networking web application where users can communicate in real time, upload photos, and share stories. The platform allows multiple users to interact through group chat and share media in an easy-to-use environment.

## Features

* 💬 Real-time group chat
* 📷 Upload and share photos
* 📖 Create and view stories
* 👥 Multiple users can chat together
* 🚪 Chat messages are cleared after users sign out
* 🎨 Simple and user-friendly interface

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Database:** MySQL

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/DILANSHIJAIN/SIMPLE_SOCIAL_NETWORK.git
   ```

   **Example:**

   ```bash
   git clone https://github.com/DILANSHIJAIN/SIMPLE_SOCIAL_NETWORK.git
   ```

2. Move the project folder to the **htdocs** directory.

   **Example (Windows):**

   ```text
   C:\xampp\htdocs\SIMPLE_SOCIAL_NETWORK
   ```

3. Start **Apache** and **MySQL** from the XAMPP Control Panel.

4. Create a MySQL database (for example, `social_network`) and import the provided `.sql` file using phpMyAdmin.

5. If required, update the database credentials in your PHP configuration file.

6. Open the project in your browser.

   **Example:**

   ```
   http://localhost/SIMPLE_SOCIAL_NETWORK/
   ```
